Descripción del Proyecto

RSE Store es un sistema web integral que revoluciona la gestión comercial de tiendas especializadas en componentes electrónicos, específicamente tarjetas para aire acondicionado inverter. El sistema combina una tienda online profesional con un panel administrativo completo que incluye control de inventario, gestión de usuarios con permisos granulares y módulo financiero con reportería avanzada.

El proyecto se desarrolló con el objetivo de digitalizar completamente las operaciones comerciales, permitiendo ventas online 24/7, control en tiempo real del stock, seguimiento financiero detallado por técnico y gestión eficiente del equipo de trabajo mediante un sistema de roles y permisos RBAC (Role-Based Access Control).

La arquitectura del sistema está diseñada como una API RESTful con separación clara entre frontend y backend, permitiendo escalabilidad y mantenibilidad a largo plazo. Incluye ~245 horas de desarrollo y más de 8,000 líneas de código profesional.

Características Principales

Carrito de Compras Profesional

Página dedicada con pasos visuales del proceso, control de cantidad, validación de stock, cálculo de totales y persistencia en localStorage.

Sistema RBAC de Permisos

Control granular con 15 permisos diferentes para productos, usuarios, finanzas y configuración. Validación en frontend y backend.

Módulo Financiero Completo

Registro de ingresos y gastos, cuadre de caja en tiempo real, reportes diarios/semanales/mensuales con calendario visual y análisis por técnico.

Panel Administrativo

Dashboard completo con gestión de productos, usuarios, inventario y finanzas. Interfaz intuitiva con navegación rápida y acciones directas.

Búsqueda y Filtros Avanzados

Búsqueda en tiempo real con resaltado de resultados, filtros por marca, tipo y rango de precio. Sistema de badges para destacar productos.

Diseño Responsive Moderno

Mobile-first con glassmorphism, gradientes cyan/magenta, animaciones suaves y diseño optimizado para todos los dispositivos.

Desafíos y Soluciones

Desafío 1: Sistema de Permisos Granular

Problema: La empresa necesitaba control detallado de qué puede hacer cada usuario sin exponer funciones sensibles.

Solución: Implementamos un sistema RBAC con 15 permisos individuales, validación en frontend y backend, y plantillas por rol para asignación rápida.

Desafío 2: Reportería Financiera Compleja

Problema: Necesitaban seguimiento detallado por técnico, tipo de servicio y método de pago en múltiples rangos de fechas.

Solución: Desarrollamos un módulo financiero con calendario visual, reportes predefinidos (diario/semanal/mensual) y análisis individualizado por técnico.

Desafío 3: Experiencia de Compra Profesional

Problema: El carrito básico no generaba confianza ni guiaba al cliente en el proceso.

Solución: Creamos una página dedicada de carrito con pasos visuales, validaciones, elementos de confianza (badges de seguridad) y confirmación por WhatsApp.

Desafío 4: Escalabilidad y Mantenibilidad

Problema: El sistema debía crecer con el negocio sin requerir reescritura completa.

Solución: Arquitectura API RESTful con separación frontend/backend, código modular y base de datos normalizada (3NF).
